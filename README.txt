- How to install

Install sfml if you don't have it
# sudo apt-get install libsfml-dev

Install cmake
# sudo apt-get install cmake

Install 3DToonChess
# mkdir build && cd build && cmake ..
# make

And run it!
# ./3DToonChess

How to run tests:
# ./tests
